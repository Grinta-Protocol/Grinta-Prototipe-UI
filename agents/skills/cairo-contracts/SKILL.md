# Cairo Contracts Skill

Enable AI agents to understand, author, and interact with Cairo / Starknet contracts.

## Key Concepts
- **Contract Structure**: Components, storage, events, and entrypoints.
- **Interfaces**: Defining and calling traits for Starknet contracts.
- **OZ v3 Patterns**: Standard implementations for accounts and tokens.
- **Storage Access**: Effective use of `StorageAddress` and `Mapping`.

## Standard Templates
- **ERC20 / ERC721**: Fundamental token standards.
- **Account Contracts**: Custom account abstraction patterns.
- **UPC & Proxy**: Upgradeable contract patterns.

## SDK Version
- Optimized for Cairo 2.x and Scarb ^2.0.0.
- Integrated with `cairo-testing` for snforge test suites.

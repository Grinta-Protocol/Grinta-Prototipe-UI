# StarkZap SDK Skill

Integrate StarkZap workflows into AI agents for native Starknet onboarding.

## Key Modules
- **StarkSDK Setup**: Initialize the multi-chain SDK.
- **Onboarding**: Manage Signer/Privy/Cartridge flows.
- **Wallet Lifecycle**: Address generation and status monitoring.
- **ERC20 Transfers**: Simple and gasless token transfers.
- **Staking Flows**: Interaction with Starknet native staking protocols.
- **Transaction Batching**: Multi-call builder for one-click workflows.

## Features
- **Sponsored Transactions**: Integrated Paymaster flows.
- **Session Keys**: Autonomous signing without repeated popups.
- **Web2 UX**: Social login and passwordless authentication.

## Supported Protocols
- **AVNU**: Liquidity and gas sponsorship.
- **Ekubo / JediSwap / 10KSwap**: Automated Market Makers.
- **Cartridge / Braavos / Argent**: Wallet providers.
